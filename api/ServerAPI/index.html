interface BanOptions {
Reason: string--

Ban reason.

PrivateReason: string--

Private ban reason.

Time: number | "inf"--

Ban time.

Moderator: number--

Moderator id.

API: boolean--

Use Roblox ban API.

Locally: boolean--

Ban locally.

ApplyToUniverse: boolean--

Apply to universe.

ExcludeAltAccount: boolean--

Exclude alt account.

}

Bans specified player.

To API ban player, you need to do the following:

API:Ban(1556153247, {
	Reason = "Bully",
	PrivateReason = "Don't unban.",
	Time = "inf",
	Moderator = 549319173,
	API = true,
	ApplyToUniverse = true,
	ExcludeAltAccount = true
})
