interface ArgumentiveLocation {
Place: string?--

The place to show.

Page: number?--

The page to show.

MaxPages: number?--

The maximum pages to show.

Arguments: any?--

The arguments to pass.

}

Works the same as :SetLocation(), but for Places that have a :Set() method.

Interface:Refresh({
	Place = "_Logs",
	Page = 1,
	MaxPages = 1,
	Arguments = {
		Type = "Chat"
	}
})
